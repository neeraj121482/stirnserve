# Text and image editing map

## Text editing

All page text is in `site-content.js`, grouped by page name.

### Home page
Edit:

```js
SNS_SITE.pages.home
```

Controls:

- homepage title
- homepage subtitle
- hero buttons
- trust pills
- homepage category card text
- favourites section text
- story teaser
- hamper CTA
- homepage reels section

### Shop All page
Edit:

```js
SNS_SITE.pages.shopAll
```

Controls `shop-all.html`.

### Snacks page
Edit:

```js
SNS_SITE.pages.snacks
```

Controls `snacks.html`.

### Sweets page
Edit:

```js
SNS_SITE.pages.sweets
```

Controls `sweets.html`.

### Pickles page
Edit:

```js
SNS_SITE.pages.pickles
```

Controls `pickles.html`.

### Gift Hampers page
Edit:

```js
SNS_SITE.pages.giftHampers
```

Controls `gift-hampers.html` including hamper cards.

### Our Story page
Edit:

```js
SNS_SITE.pages.ourStory
```

Controls `our-story.html`, including all story paragraphs.

### Our Promise page
Edit:

```js
SNS_SITE.pages.ourPromise
```

Controls `our-promise.html`, including all promise cards and FSSAI text.

### Payment & Orders page
Edit:

```js

```

Controls `payment-orders.html`.

### Contact page
Edit:

```js
SNS_SITE.pages.contact
```

Controls `contact.html` page title, subtitle, section text and contact details.

## Product text and prices

Edit `products.js`.

This controls:

- product page name
- product page description
- product card text
- ingredients
- taste profile
- storage
- best with
- sizes
- was price
- now price
- payment link
- product image base name

## Image editing

### Page hero images

Folder:

```text
assets/page-images/
```

Base names:

- `home-hero`
- `shop-all-hero`
- `snacks-hero`
- `sweets-hero`
- `pickles-hero`
- `gift-hampers-hero`
- `our-story-hero`
- `our-promise-hero`
- `payment-orders-hero`
- `contact-hero`

Upload PNG, WEBP, JPEG or JPG with the same base name.

Example:

```text
assets/page-images/shop-all-hero.png
```

or

```text
assets/page-images/shop-all-hero.jpg
```

### Homepage category images

Folder:

```text
assets/category-images/
```

Base names:

- `savory`
- `sweets`
- `pickles`
- `hampers`
