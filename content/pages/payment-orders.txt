How online payment works here

Section Text:
GitHub Pages is static hosting, so online payments should be handled through safe external payment links.

Cards:
WhatsApp Ordering | Every product page and cart can send the order details directly to WhatsApp.
Payment Links | Create Razorpay or other payment links for each size and paste them in products.js.
No Secret Keys | Do not put payment gateway secret keys inside HTML, JavaScript or GitHub Pages.
