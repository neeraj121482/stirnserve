- `assets/product-images/namak-para`
- `assets/product-images/sada-mathri`
- `assets/product-images/methi-mathri`
- `assets/product-images/lehsuni-bites`
- `assets/product-images/sakar-para`
- `assets/product-images/besan-kesar-ladoo`
- `assets/product-images/atta-dryfruit-ladoo`
- `assets/product-images/lehsuwa-ka-achar`
- `assets/product-images/ker-ka-achar`

Upload any of these extensions: `.png`, `.jpg`, `.jpeg`, `.webp`. PNG is checked first. If both PNG and JPG exist with the same base name, PNG wins.
